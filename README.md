# Radio Names DB

Приложение для управления базой названий радиостанций для Belgee X50.

Работает в паре с модулем **RadioNameHook** (LSPosed).

## Что умеет

- 📍 Загружать готовые регионы (Москва, СПб) одним кликом
- ➕ Добавлять станции вручную (FM/AM)
- ✏️ Редактировать названия
- 🗑 Удалять станции
- 💾 Сохранять в `/sdcard/RadioNames/stations.json` — это файл который читает LSPosed модуль

## Использование

1. Установи приложение
2. Дай разрешение на доступ к файлам (откроется автоматически)
3. Нажми **"📍 Регион"** → выбери Москву или СПб
4. Можешь редактировать названия и добавлять свои
5. Нажми **"💾 Сохранить"**
6. LSPosed модуль автоматически подхватит новую базу

## Формат файла

`/sdcard/RadioNames/stations.json`:

```json
{
  "fm": {
    "103.4": "Русское Радио",
    "104.2": "Авторадио"
  },
  "am": {
    "675": "Маяк"
  }
}
```
